# USD/KRW 환율 텔레그램 알람 봇

매일 오전 08:50 KST에 자동으로 실행되어 USD/KRW 환율을 분석하고 텔레그램으로 거래 판단을 발송하는 자동화 봇입니다.

## 📊 핵심 기능

### 1. 일일 자동 브리핑 (08:50 KST)
- **NDF 갭 계산**: (오늘 시가 - 전날 종가) / 전날 종가 × 100
- **판단 로직**:
  - 갭 마이너스 (-) → **매수** 추천 (저가 진입 기회)
  - 갭 0~+0.3% → **관망** (추가 신호 대기)
  - 갭 +0.3% 초과 → **스킵** (고가 형성)
  - B패턴 2일 연속 → **매도** 신호

### 2. 장중 실시간 알람
| 시간 | 기능 | 내용 |
|------|------|------|
| 09:00 | 관망 재확인 | 추가 신호 확인 |
| 13:00 | 중간 점검 | 포지션 보유 중 현황 확인 |
| 14:50 | 매도 타이밍 | A패턴 매도 신호 검사 |
| 수시 | 급변 감지 | ±0.5% 이상 환율 변동 시 즉시 알람 |

### 3. 데이터 관리
- **Yahoo Finance 연동**: 실시간 시간별 환율 데이터 수집
- **CSV 아카이빙**: 매일의 판단 결과와 NDF 갭을 자동 기록

## 🚀 설치 및 실행

### 1단계: 필수 라이브러리 설치
\\\ash
pip install -r requirements.txt
\\\

### 2단계: 텔레그램 봇 설정
1. Telegram에서 @BotFather와 대화 시작
2. /newbot 명령으로 새 봇 생성
3. 받은 **BOT_TOKEN** 복사

### 3단계: 채팅 ID 확인
1. 생성한 봇과 대화 시작
2. https://api.telegram.org/bot<BOT_TOKEN>/getUpdates 접속
3. 응답의 \chat\ → \id\ 값 복사

### 4단계: config.py 설정
\\\python
BOT_TOKEN = "여기에입력"  # 위에서 받은 BOT_TOKEN 입력
CHAT_ID = "여기에입력"    # 위에서 받은 CHAT_ID 입력
\\\

### 5단계: 봇 실행
\\\ash
python forex_alarm.py
\\\

## 🐛 문제 해결

### \ModuleNotFoundError\ 에러
\\\ash
pip install -r requirements.txt --upgrade
\\\

### 텔레그램 메시지가 오지 않음
- config.py에서 BOT_TOKEN, CHAT_ID 재확인
- 콘솔에 \[경고] BOT_TOKEN 또는 CHAT_ID가 설정되지 않았습니다.\ 확인

### Yahoo Finance 데이터 오류
- 인터넷 연결 확인
- VPN 사용 시 비활성화
- 시간대가 정확한지 확인
